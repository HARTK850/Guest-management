טיפ: השתמש בעוזר הקולי

לחץ על כפתור 🎤 לתוספת נתונים בקול מהיר וקל

🎤 עוזר קולי

הדרך הקלה ביותר להוסיף נתונים בקול! לחץ על כפתור "עוזר קולי" בכל מקום.

  • הוספת שבת בקול: אמור את התאריך העברי והפרשה
  • הוספת אורח בקול: אמור את שם האורח ומספר הנפשות
  • הוספת משימה בקול: אמור כל משימה שצריך לעשות
  • הוספת הערות בקול: הכתב הערות כלליות על השבת
  • פקודות קוליות: אמור "הבא" או "סיימתי" לסיום

🔧 פתרון בעיות נפוצות

❓ העוזר הקולי לא עובד

בדוק:

  • האם דפדפן שלך תומך בקול? (Chrome, Edge, Firefox - כן)
  • האם מיקרופון מחובר ופועל?
  • האם אישרת גישה למיקרופון בדפדפן?
  • האם שרת Python פועל על localhost:5000?
  • נסה לרענן את הדף (Ctrl+R או Cmd+R)

❓ "שגיאת רשת" בעוזר הקולי

פתרון:

  • ודא שרת Python פועל: python3 voice_server.py
  • בדוק שהשרת אמור להאזין על פورט 5000
  • בדוק את Console של הדפדפן (F12 → Console) לשגיאות
  • ודא שה-Google Speech API זמין בשרת שלך

❓ כיצד להפעיל את שרת התימלול

בשורת הפקודה:

pip install -r requirements.txt
python3 voice_server.py
